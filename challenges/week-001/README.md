# Week 001: Build a CLI Calculator

## Difficulty: Beginner

## Problem

Build a command-line calculator that:
1. Takes two numbers and an operator as input
2. Supports addition (+), subtraction (-), multiplication (*), division (/)
3. Handles division by zero gracefully
4. Displays the result

## Example

```
Input:  5 + 3
Output: 8

Input:  10 / 0
Output: Error: Division by zero

Input:  7.5 * 2
Output: 15.0
```

## Bonus (Optional)

- Support chained operations: `5 + 3 * 2`
- Add history: show the last 5 calculations
- Support exponentiation: `2 ^ 8`

## Constraints

- Handle invalid input (non-numbers, unsupported operators)
- Print a helpful error message, do not crash
