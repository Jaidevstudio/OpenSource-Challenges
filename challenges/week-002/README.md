# Week 002: Palindrome Checker with Unicode Support

## Difficulty: Beginner

## Problem

Build a program that checks if a given string is a palindrome.

Requirements:
1. Case-insensitive comparison
2. Ignore spaces and punctuation
3. Support Unicode characters (Hindi, Bengali, Arabic, etc.)
4. Return true/false with the cleaned string

## Example

```
Input:  "racecar"
Output: true (racecar)

Input:  "A man a plan a canal Panama"
Output: true (amanaplanacanalpanama)

Input:  "Hello World"
Output: false (helloworld)

Input:  "नमन"
Output: true (नमन)
```

## Bonus (Optional)

- Check if a number is a palindrome without converting to string
- Find the longest palindromic substring in a given string
- Support checking multiple strings from a file

## Constraints

- Do not use any built-in palindrome functions
- Handle empty strings and single characters
